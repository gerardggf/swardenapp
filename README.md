# swarden

Gestor de contrasenyes Open Source E2EE desenvolupat amb Flutter.


## Compil·la el projecte

Executa:
- flutter pub get
- flutter pub run build_runner build
- flutter pub run slang

## Testing

Executa:
- flutter test
- flutter drive --target=test_driver/app.dart
