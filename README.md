# Swarden

Gestor de contrasenyes Open Source E2EE per Android i iOS desenvolupat amb Flutter.


## Set up

Per compil·lar l'aplicació, des de l'arrel del projecte executa:
1. ```flutter pub get```
2. ```flutter pub run build_runner build```
3. ```flutter pub run slang```

## Testing

Per executar les proves unitàries i de widget:
- ```flutter test```

Per executar les proves d'integració:
- ```flutter drive --target=test_driver/app.dart```
