# Contribuir a Swarden

Gràcies per voler contribuir a Swarden! Aquest document explica com reportar problemes, proposar millores i enviar canvis.


## Com col·laborar

- Reporta bugs obrint una "Issue" amb els passos per reproduir el problema, el comportament esperat i l'entorn.
- Proposa funcionalitats.
- Revisa les PRs (Pull Requests).


## Flux de treball de desenvolupament

1. Fes un fork del repositori i crea una branca des de main.
2. Fes canvis petits i enfocats.
3. Executa els tests assegurant que tots passen.
4. Actualitza el README.
5. Escriu els commits clars. Recomanació: https://www.conventionalcommits.org.
6. Obre una Pull Request.

## Estàndards de codi

- Segueix els linters definits al repositori.
- Mantén la compatibilitat i no trenquis APIs públiques sense justificació.
- Respecta la nomenclatura i els comentaris.


## Llicència de contribucions

Al enviar un PR, acceptes que la teva contribució es publiqui sota la llicència del projecte (Apache 2.0). No aportis contingut que no puguis llicenciar.


Gràcies per fer Swarden millor!
